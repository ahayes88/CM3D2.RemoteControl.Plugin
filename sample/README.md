* all_*.bat
  * puttyに付属のplink.exeを利用して、バッチファイルからCM3D2にコマンド文字列を送信するサンプル。
  * 実行前に`PLINK`を環境に合わせて書き換えてください。
  * 展開した`voice`や`motion`ディレクトリ内に置いて実行してください。
  * カレントディレクトリ内にあるファイルを片っ端から実行します。
  * Ctrl+Cで停止、何かキーを押すと次のファイルに進みます。
* simple_command.pl
  * PerlからIO::Socketを利用してコマンドを送信するサンプル
  * 5秒ごとにランダムなMOTIONを実行します
* check_newmail.pl
  * メールサーバにIMAP接続を行い、新着メールが届いていたらコマンドを実行する
  * メールサーバ設定(MAIL_*)を変更してからお使いください
